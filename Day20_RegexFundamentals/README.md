# What is Regex?

Regex (Regular Expression) is a pattern matching language used for:

- Searching Text
- Validating Data
- Extracting Information
- Replacing Content

---

# Important Functions

## search()

Searches the entire string.

## match()

Checks only the beginning.

## fullmatch()

Matches the complete string.

## findall()

Returns all matches.

## finditer()

Returns match objects one by one.

## split()

Splits text using regex.

## sub()

Replaces matched text.

---

# Character Classes

| Pattern | Meaning |
|----------|----------|
| \d | Digit |
| \D | Non-Digit |
| \s | Whitespace |
| \S | Non-Whitespace |
| \w | Word Character |
| \W | Non-Word Character |

---

# Quantifiers

| Pattern | Meaning |
|----------|----------|
| * | 0 or More |
| + | 1 or More |
| ? | 0 or 1 |
| {n} | Exactly n Times |
| {n,m} | Between n and m Times |

---

# Anchors

$ = End
