# Day 4 - Advanced File Handling

## Aim

To study Advanced File Handling in Python using CSV files, JSON files, Log Files, and Data Processing.

---

## Theory

### CSV (Comma Separated Values)

CSV files are used to store tabular data in rows and columns.

Example:

Name,Age,City
Shubham,20,Kolhapur
Rahul,21,Pune

Applications:
- Student Records
- Employee Data
- Reports
- Database Exports

---

### JSON (JavaScript Object Notation)

JSON is used to store structured data in key-value format.

Example:

{
    "name": "Shubham",
    "age": 20,
    "city": "Kolhapur"
}

Applications:
- APIs
- Cloud Services
- Configuration Files
- Data Exchange

---

### Log Files

A log file records events occurring in a program.

Example:

04-06-2026 Login Successful
04-06-2026 File Uploaded
04-06-2026 Logout

Applications:
- Monitoring Systems
- Error Tracking
- Security Auditing
- Cloud Operations

---

### Data Processing

Data Processing means converting raw data into useful information.

Example:

Input Data:
Name,Marks
Shubham,85
Rahul,78
Priya,92

Processed Data:
Total Marks = 255
Average Marks = 85

Applications:
- Analytics
- Reports
- Monitoring Systems
- Cloud Metrics
