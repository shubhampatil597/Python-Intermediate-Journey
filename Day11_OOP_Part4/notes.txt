DAY 11 NOTES

Inheritance

Definition:
Allows one class to inherit properties and methods from another class.

Benefits:

1. Code Reusability
2. Easy Maintenance
3. Scalability
4. Better Design

Parent Class

Provides functionality.

Child Class

Inherits functionality.

Single Inheritance

One Parent
One Child

Example:

Animal → Dog

Multilevel Inheritance

Parent → Child → Grandchild

Example:

Animal → Dog → Puppy

Hierarchical Inheritance

One Parent
Many Children

Example:

Animal → Dog
Animal → Cat
